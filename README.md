Rock Paper Scissors

Simple Rock–Paper–Scissors game in JavaScript for The Odin Project. 
Has interactive UI.

How to Play

Open index.html in your browser.

Open the developer console.

Run:

playGame()


Enter rock, paper, or scissors when prompted.

First to 5 wins takes the game.

Features

Random computer choice

Input validation (rock/paper/scissors only)

Score tracking and winner announcement
